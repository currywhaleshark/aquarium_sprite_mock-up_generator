물고기 영법 조사와 모션 에디터 적용 노트

이 문서는 물고기/가오리 스프라이트 생성기의 모션 프로파일을 설계하기 위한 참고 자료다. 학술 분류를 그대로 재현하기보다는, 현재 Godot 프로토타입의 링 기반 몸통과 지느러미 파라미터에 바로 옮길 수 있는 조절 포인트를 정리한다.

작성일: 2026-05-31 대상: FishRig / BodyProfile / Motion Settings 핵심: swim mode preset + 미세 조정 슬라이더

요약

1. 큰 축은 BCF와 MPF

BCF는 body and caudal fin, 즉 몸통과 꼬리지느러미로 추진한다. MPF는 median and paired fins, 즉 등/뒷/가슴/배지느러미 중심 추진이다. BCF는 추력과 가속, MPF는 저속 기동과 정밀 제어에 유리한 경향이 있다.

2. 영법은 카테고리보다 연속체

전통 분류는 유용하지만 절대적인 규칙은 아니다. 최근 비교 연구는 다양한 종의 정상 유영이 생각보다 비슷한 2D midline kinematics를 공유한다고 보고한다. 따라서 UI에는 딱딱한 분류보다 조절 가능한 템플릿으로 넣는 편이 좋다.

3. 지금 필요한 것은 모션 모드 프리셋

body_wave_amount, body_wave_start, body_wave_falloff, tail_sway_multiplier, fin_flap_amount 등을 한 묶음으로 바꾸는 swim_mode가 필요하다.

시각 자료 1: BCF 영법의 몸통 파동 분포

아래 도식은 머리에서 꼬리로 갈수록 lateral amplitude가 어디서 시작하고 얼마나 커지는지를 단순화한 것이다. 실제 생물은 더 복잡하지만, 에디터 파라미터로는 이 정도 모델이 충분히 실용적이다.

시각 자료 2: BCF와 MPF의 추진 부위

일반 물고기형은 BCF를 기본으로 잡고, 복어/개복치/박스피시 계열은 MPF 성분을 강하게 섞는 것이 좋다. 가오리형은 별도 disc/wing section 구조로 MPF 계열을 다루는 편이 자연스럽다.

영법별 적용 표

모드 대표 이미지 움직임 특징 에디터 파라미터 방향
anguilliform 장어, 칠성장어형 머리 가까운 앞쪽부터 전신에 파동이 들어간다. 유연하고 기동성 강함. body_wave_start 0.00-0.08, body_wave_amount 높게, falloff 낮게.
subcarangiform 송어, 일반 어류 앞몸통은 비교적 안정적이고 중후반부터 진폭이 커진다. 기본값. body_wave_start 0.12-0.25, falloff 0.6-0.9.
carangiform 고등어류, 빠른 유선형 어류 몸통 후방 1/3과 꼬리자루 중심. 꼬리가 주 추진원. body_wave_start 0.35-0.50, tail_sway_multiplier 중간 이상, 머리 안정.
thunniform 참치, 일부 상어형 몸통 대부분은 단단하고 꼬리자루와 높은 aspect ratio 꼬리가 중심. body_wave_amount 낮음, body_wave_start 높음, 꼬리 각도는 과장보다 빠르고 절제.
ostraciiform 박스피시, 트렁크피시 몸통이 거의 뻣뻣하고 꼬리지느러미가 진자처럼 움직인다. 몸통 파동 거의 0, 꼬리만 작은 각도로 반복. 속도보다 정밀함.
tetraodontiform 복어, 개복치 계열 몸통 굴곡보다 등/뒷/가슴지느러미 의존도가 크다. fin_flap_amount 높게, body_wave_amount 낮게, 꼬리 추가 스윙 낮게.
rajiform/mobuliform 가오리, 쥐가오리 커진 가슴지느러미 또는 disc/wing이 파동 또는 플랩한다. FishRig보다 RayRig 전용 section/wing 파라미터로 분리.

구현 제안

권장: Motion Settings에 모든 세부 슬라이더를 늘리는 대신, 먼저 swim_mode 드롭다운을 추가한다. 드롭다운은 여러 motion 값을 한 번에 적용하고, 현재 노출된 세부 슬라이더는 그 뒤의 미세 조정용으로 남긴다.

  "motion_profile": {
    "swim_mode": "general",
    "swim_speed": 1.0,
    "global_sway_amount": 10.0,
    "phase_delay": 0.62,
    "body_wave_amount": 0.35,
    "body_wave_start": 0.16,
    "body_wave_falloff": 0.75,
    "tail_sway_multiplier": 1.2,
    "tail_fin_extra_swing": 0.45,
    "fin_flap_amount": 6.0,
    "fin_yaw_follow_strength": 0.25,
    "idle_bob_amount": 0.025
  }
}

현재 코드에 바로 연결할 항목

문서 항목 현재 또는 권장 코드 항목 비고
몸통 파동 시작 위치 body_wave_start 머리를 안정시키고 싶은 어종일수록 높인다.
몸통 파동 분포 body_wave_falloff 값이 클수록 후방에 몰린다.
꼬리자루 증폭 tail_sway_multiplier 꼬리 뿌리와 꼬리자루 움직임. 너무 높이면 파츠가 과장된다.
꼬리지느러미 추가 회전 tail_fin_extra_swing 권장 현재 상수로 들어간 값을 프로파일 값으로 빼는 것이 좋다.
비꼬리 지느러미 yaw 추적 fin_yaw_follow_strength 권장 현재 상수로 들어간 값을 모션/핀 프로파일 값으로 옮기면 어종별 제어가 쉽다.
복어/개복치식 지느러미 추진 fin_flap_amount, dorsal/anal paired motion 추후 등/뒷지느러미 별도 flap phase를 넣으면 표현력이 커진다.

디자인 판단

  • 영법 모드는 생물학적 정답이 아니라 아트 디렉션용 시작점으로 다룬다.
  • 꼬리지느러미 각도는 크면 곧바로 장난감처럼 보인다. 꼬리자루는 움직여도 끝 지느러미의 추가 회전은 절제한다.
  • 복어류는 꼬리만 크게 흔드는 것보다 몸통 고정 + 등/뒷/가슴지느러미 플랩으로 가야 설득력이 있다.
  • 가오리형은 fish body ring을 억지로 쓰기보다 disc/wing section 파동으로 따로 가는 편이 좋다.
